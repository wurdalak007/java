Следующие запросы реализованны в классе solver.Requester


1) Вывести Top-10 самых активных пользователей за последние A дней. Пользователя идентифицируем по IP. A - параметр.
2) Написать запрос выбирающий регионы, в которых количество посещений выше среднего количества посещений по всем регионам.
3) Выбрать количество посещений женщин и мужчин для данного дня. День - параметр запроса.
4) Выдать самые популярные ресурсы среди пользователей России младше A лет за последние B дней в логах. A, B - параметры.
5) Из каких регионов было больше всего посещений сайта lenta.ru последние N дней. N - параметр.
6) Написать запрос на добавление нового посещения в таблицу логов.
7) Написать запрос на удаление “старых” (старше некоторой даты, кот. является параметром) логов из базы.
8) Запрос на обновление: представим, что все сайты начиная с дня N в логах переехали с домена .ru на домен .com. Нужно отредактировать логи пост-фактум чтоб они соответствовали действительности. N-параметр.
